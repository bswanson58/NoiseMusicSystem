Credits

Underlying this system are many supporting libraries, efforts and ideas. Credit has been split into software and graphical sections.

Software

BASS Audio - provides the playback of all audio files.

Caliburn.Micro - provides intuitive publish/subscribe mechanism and a better ObservableCollection class.

CefSharp - underlies the documentation you are currently reading.

DynamicData - underlies the playback queue.

EntityFramework - is used for the primary access to data stored in the local SQL Server database

GongSolutions - adds drag and drop of items in the user interface, particularly the playback queue.

Lucene.Net - support searching throughout the music database.

MahApps.Metro - provides the overall appearance of the application.

Prism - aids the overall construction of the application with Modules, Regions, view model connection and DelegateCommands.

RavenDB - is used to store metadata such as artist biographies and band information from various providers. The version used is embedded and no longer supported.

ReactiveUI - used selectively to support advanced UI actions.

Serilog - is used to collect and store logging information throughout the system.

ReFit - used to support iOS and Android clients to the desktop

Stateless - forms the basis for the playback queue operation and it's many states.

Taglib# - reading of tags from mp3 files.

TinyIpc - communicates information to support the 'companion apps' interaction.

Unity - Inversion of control, used extensively to construct the many class structures.

Graphics

All of the graphics within the application are vector based to allow scaling to any size while maintaining their detail. One of the best sites for vector based artwork is the Noun Project. The SVG supplied artwork files may be loaded into Adobe Illustrator and then exported to XAML files with a plugin by Mike Swanson called XAML Export.
